Using Maven Artifacts

If you use Maven as the build system (or one that is compatible with the Maven Repository), you can declare dependencies as 

   
    org.picontainer.logging
    picocontainer-logging
    [version]
specifying the version. Declaring this dependency will pull in all other required dependencies (ie transitive dependencies). Note that picocontainer-logging has no transitive dependency on PicoContainer.
