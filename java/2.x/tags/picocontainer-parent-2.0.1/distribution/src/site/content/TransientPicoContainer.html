TransientPicoContainer is a for short term needs for object instantion, where the container is going to be de-referenced shortly after use.

Here is an example of use: 

public Foo makeSuitableFoo() {
    MutablePicoContainer temp = new TransientPicoContainer(someParentContainer);
    temp.addComponent(somethingElseThatMightBeNeeded);  // by instance
    temp.addComponent(Foo.class, FooImpleThatNeedsTheAbove.class); // by type

}
