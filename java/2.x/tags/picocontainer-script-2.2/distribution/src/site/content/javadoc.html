Script Core

The Core module is not tied to any scripting language - it depends only on PicoContainer and the JDK itself. It does provide support for XML.

Script BeanShell

The BeanShell module provides support for BeanShell scripting.

Script Groovy

The Script Groovy module provides support for Script Groovy scripting.

Script JRuby

The Script JRuby module provides support for Script Script JRuby scripting.

Script Jython

The Script Jython module provides support for Jython scripting.

Script Rhino

The Script Rhino module provides support for Rhino scripting.
