Web Core

The Core module is not tied to any web framework - it depends only on PicoContainer and optionally on PicoContainer Script.

Web JSF

The JSF module provides support for JSF.

Web Struts

The Struts module provides support for Struts 1.x.

Web Struts2

The Struts2 module provides support for Struts 2.x.

Web WebWork

The WebWork module provides support for WebWork 1.x.

Web WebWork2

The WebWork2 module provides support for WebWork 2.x.
