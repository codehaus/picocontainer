Using Maven Artifacts

If you use Maven as the build system (or one that is compatible with the Maven Repository), you can declare dependencies as 

  
      org.picocontainer
      picocontainer[-gems]
      [version]

specifying the version. Declaring this dependency will pull in all other required dependencies (ie transitive dependencies). Note that PicoContainer core artifactId is simply "picocontainer".
