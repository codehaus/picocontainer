Why PicoContainer Script?

The main scope of PicoContainer Script to provide a scripting layer on top of PicoContainer, adding the following features:

  • Classloader (classpath) management, including programmable permissions
  • Class name based composition (via reflection)
  • Meta-data and script language support: XML, Groovy, Beanshell, Jython, Rhino (Javascript)

It is important to stress that there is no "official" script or meta-data language.
