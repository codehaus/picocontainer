PicoContainer-Script 2.0.1 released - Mar 15, 2009

New (since 2.0):

  • XStreamContainerBuilder supports the same attributes as XMLContainerBuilder

Changes (versus 2.0):

  • Upgraded to Groovy 1.6
  • Upgraded to Apache CLI 1.1
  • Upgraded to XStream 1.3.1
  • Small bugs fixed

PicoContainer-Script 2.0 released - June 27, 2008

This is the first release of a PicoContainer Script. It has a 2.0 release number because it is tracking the release cycle of PicoContainer itself, and cannot rely on PicoContainer 1.x. Formerly these script components were part of the NanoContainer project.
