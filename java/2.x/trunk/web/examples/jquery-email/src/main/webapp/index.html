~ About This Web-App ~

This app uses Pico Web Remoting. It is a fork of web-app made for an AlphaWorks article:
"Working with jQuery, Part 1: Bringing desktop applications to the browser".
You can mimick what JQuery invokes with these URLs:

  • pwr/Inbox/read?msgId=1 - read a message (and have it marked as read on the backend)
  • pwr/Inbox/messages?userId=1 - see the contents of the inbox
  • pwr/Sent/messages?userId=1 - see the contents of the sent box
  • pwr/Sent/send?to=fred&subject=hello&message=goodbye - see send an email (this is normally a POST operation
