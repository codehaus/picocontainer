Here's what to add to web.xml


    <listener>
        <listener-class>nanocontainer.servlet.lifecycle.ApplicationLifecycleListener</listener-class>
    </listener>

    <listener>
        <listener-class>nanocontainer.servlet.lifecycle.SessionLifecycleListener</listener-class>
    </listener>

    <filter>
        <filter-name>nanocontainer</filter-name>
        <filter-class>nanocontainer.servlet.lifecycle.RequestLifecycleFilter</filter-class>
    </filter>

    <filter-mapping>
        <filter-name>nanocontainer</filter-name>
        <url-pattern>/*</url-pattern>
    </filter-mapping>

Nanocontainer will then read three files in the WEB-INF directory that contain components to be instantiated by the container : components-application.xml, components-session.xml and components-request.xml.

From the web-app, you would typically want to access the request scope container (which is linked to session and application containers) with the following code:

Container container = new WebContainer(request);
