PicoContainer (the project) ships with a few containers.

DefaultPicoContainer

The basic PicoContainer.  No frills.  This one has component registration by class.

CompositePicoContainer

This one takes an array of PicoContainers to build a contrived single container. You may use that container as the parent to others. Thus trees of containers are not the only design.

Any Pico-style container that honors the PicoContainer interface can be eligible for inclusion in the composite. These could be DefaultPicoContainer or any other the Nano containers (sister project), or those from third parties.
