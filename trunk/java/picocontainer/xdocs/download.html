If you're using Maven to build a project that needs picocontainer, you should tell maven to use Codehaus' repository in addition to the one at Ibiblio. Put the follwing in your project's project.properties file:
maven.repo.remote = http://www.ibiblio.org/maven/,http://dist.codehaus.org/
For CVS access, see Source Repository.
