Anonymous CVS Access

This project's CVS repository can be checked out through anonymous (pserver) CVS with the following instruction set. When prompted for a password for anonymous, simply press the Enter key.

cvs -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/picocontainer login

cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/picocontainer co java ( Java PicoContainer, NanoContainer plus extras )
cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/picocontainer co dotnet ( .Net Pico Container )
cvs -z3 -d :pserver:anonymous@cvs.codehaus.org:/cvsroot/picocontainer co ruby ( Ruby Pico Container )

Updates from within the module's directory do not need the -d parameter.

Developer CVS Access via SSH

Only project developers can access the CVS tree via this method. SSH1 must be installed on your client machine. Substitute name with the proper value. Enter your site password when prompted.

export CVS_RSH=ssh

cvs -z3 -d :ext:username@cvs.codehaus.org:/cvsroot/picocontainer co module-as-above

Remember to replace 'username' with your actual username on cvs.codehaus.org.

CVS Access behind a firewall

For those developers who are stuck behind a corporate firewall, CVSGrab can use the viewcvs web interface to checkout the source code.
